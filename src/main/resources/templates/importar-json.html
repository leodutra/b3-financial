Endpoints.

Todos os endpoints que a API disponibiliza para você usar :). Faça bom proveito!

# Tipo Endpoint
1 Ação /statusinvest/acoes/{ticket}
2 Ação /statusinvest/acoes/{ticket}/{indicador}
3 FII /statusinvest/fundos-imobiliarios/{ticket}
4 FII /statusinvest/fundos-imobiliarios/{ticket}/{indicador}
5 ETF /statusinvest/etfs/{ticket}
6 Todos /statusinvest/?tickets=X,Y,Z

Indicadores disponíveis

São todos os indicadores que a aplicação recupera da StatusInvest.

Você também pode utilizar nos serviços acima, mas somente para os endpoints que possuem parâmetro {indicador}.

# Chave Indicador
1 VALUE Valor por ação atualizado de Mercado
2 PL P/L - Preço atual / Lucro por ação (LPA)
3 LPA Lucro líquido / Nº de ações
4 PVP P/VP - Preço atual / Valor patrimonial por ação (VPA)
5 VPA Patrimônio líquido / Nº de ações
6 DY Dividend Yield - % dos últimos 12 meses.

*O Valor não é um indicador, é mais caso você precise somente do valor mesmo :)

Dicas

  • Todos os tickets são validados com os existentes na B3, mas isso foi inserido manualmente. Portanto, caso esteja utilizando a API para algum ticket muito novo, você deve me avisar para incluí-lo na aplicação, pois pode retornar erro de ticket inválido.
  • Indicadores que não forem encontrados na página da StatusInvest serão retornados com valor 0,00.
  • O uso incorreto de um endpoint poderá retornar 404 ou 500 como erro.
  • O volume excessivo de chamadas para a API pode retornar em um delay na resposta de até 15 segundos, conforme bloqueio da StatusInvest
